# 物語・エンディング・神機マキナ

実装: `src/data/lore.ts`（文言の単一ソース）、進行は `src/store/gameStore.ts`、
表示は `src/components/BattleScreen.tsx`。

## Vision

Dice Ex Machina は「ダイス版 Diablo」。終わりのない数字を追うゲームではない。
**1000階に終わりがあり、その先はプレイヤー自身の物語**になる。プレイヤー自身は成長せず、
強くなるのは装備・ビルド・知識。大切にするのはテンポ／装備更新／ビルド／世界の変化／
長い旅の終わりの寂しさ。

## 1000階エンディング（DEUS EX MACHINA 撃破）

スタッフロール（黒背景・白文字）。**飛ばせない／一度きり／EXTRAから再視聴不可**。

```
記録の終端を確認しました。
これより先の領域は、想定されていません。
あなたはここで終わることもできます。
あるいは……
全てを失い、再び始めますか？
```

選択肢: **▶ はい（強くてニューゲーム）** ／ **いいえ（Endless Abyssへ）**

### YES — 強くてニューゲーム

全て失う（レベル・装備・ゴールド）。アーティファクトと最高到達記録は引き継ぐ。称号
**Makina-0001** を獲得し、**神機マキナ** を付与。

```
初期化完了 / 識別番号 Makina-0001 / 再起動します / 神機マキナを付与する。
```

### NO — Endless Abyss

1001階以降へ。以後、最高到達階を更新するたびに一度だけ物語が挿入される:

| 階 | 台詞 |
| --- | --- |
| 1050 | まだいましたか。 |
| 1100 | 私は、あなたはここで終わるものだと思っていました。 |
| 1150 | それでもなお、先へ進むのですね。 |
| 1200 | 理解できません。ですが、嫌いではありません。 |
| 1250 | ……それでは。続きを始めましょう。**ここで神機マキナを授与**。以後システムメッセージは発生しない。 |

1250階以降はゲームが用意した物語ではなく、**プレイヤー自身の物語**となる。

## 神機マキナ（🌈✦✦✦✦ Unique）

世界に一本だけ。YESルートでも、NOルート（1250階）でも取得する。

- 売却不可・分解不可・Modifierなし・Ancient化しない・Mythic化しない（`unique`/`noSell`/`noModifier`）。
- 性能: 現在実装されている理論値最強ドロップ武器の **92%**（`makinaAttackValue()`）。
- 固有能力 **Complete**: 全ての出目（1〜6）を通常攻撃に変える。

> 「終わりを見届けた者だけが手にすることを許された武器。」
> 「何度失われても、必ずここへ帰ってくる。」
> 「最強ではない。ただ、もう迷う必要はない。」

## 進行フラグ（`Progress`）

- `endingSeen` … エンディング視聴済み（YES/NOどちらかを選んだ）。
- `ngPlus` … 強くてニューゲーム回数。
- `makinaGranted` … 神機マキナ付与済み（重複付与しない）。
- `claimedEndlessMessages` … 表示済みのEndless台詞の階層。
