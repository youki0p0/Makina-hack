# セット装備

実装: `src/data/sets.ts`（定義と効果計算 `computeSetEffects`）、適用は `src/store/gameStore.ts`。

セット装備は Dice Ex Machina の重要要素。単なる数値アップではなく **ビルドを作る存在**。
6スロットに同じセットの装備を揃えると、2 / 4 / 6 部位で効果が段階的に解放される。

## 賭博師（🎲 gambler）

| 部位 | 効果 |
| --- | --- |
| 2 | リロール +1 |
| 4 | 出目1が2（通常攻撃）として扱われる |
| 6 | 出目6が2回発動する |

## 吸血鬼（🦇 vampire）

| 部位 | 効果 |
| --- | --- |
| 2 | 与ダメージの10%を吸血 |
| 4 | 出目4以上で追加吸血（+15%） |
| 6 | 全ての攻撃が30%吸血 |

## 処刑人（🪓 executioner）

| 部位 | 効果 |
| --- | --- |
| 2 | 出目5・6の威力上昇（+30%） |
| 4 | 攻撃時に追撃（+1ヒット） |
| 6 | 敵HP15%以下を即死（ボスを除く） |

## 神託（🔮 oracle）

| 部位 | 効果 |
| --- | --- |
| 2 | リロール時にHP回復 |
| 4 | 出目6の威力上昇（+60%） |
| 6 | ダイスを2個振り、高い方を使う |

## 実装メモ（決定論ダイスエンジンとの整合）

本作の出目は値で確定する決定論エンジンのため、セット効果は次の方針で適用している:

- **ダイス面書き換え**で表せるもの（賭博師4部位 1→2）は `diceModifiers` として
  `applyEquipmentModifiers` の最後段で合流。
- それ以外（吸血・高目ボーナス・追撃・即死・6の2回発動・リロール回復・2個振り）は
  `computeSetEffects` が返すフラグ/数値を `confirm()` / `reroll()` / `rollWithLuck()` で
  ダメージ・回復・出目に**加算的に**反映する。
- 「処刑（即死）」はボスには適用しない（ボス戦の興を削がないため）。
- ステータス系（賭博師2部位のリロール+1）は `passiveBonus()` に合算。

発動中のセット効果は装備画面（`EquipmentPanel`）に一覧表示され、図鑑（/collection の
「セット」タブ）で各セットの効果と収集率（X/6）を確認できる。
